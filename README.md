# POS System

## Project Setup
To set up the project, run the following command:
```
npm install
```

### Development
To compile and hot-reload for development, use:
```
npm run serve
```

### Production
To compile and minify for production, use:
```
npm run build
```

### Linting
To lint and fix files, use:
```
npm run lint
```
